About

These settings control the default ports used when opening AEM tools on localhost.

  • Author Port (4502): Used for CRXDE, Package Manager, Config Manager, etc.
  • Publish Port (4503): Used when opening publish instance or publish login

Note: These settings only affect localhost connections. AEM Cloud instances use their default configurations.
