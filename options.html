About

Configure multiple AEM projects with different settings. The extension automatically detects which project to use based on the current URL.

  • URL Pattern: Matches the hostname. Use * as wildcard (e.g., *.adobeaemcloud.com matches all AEM Cloud URLs)
  • Author/Publish Ports: Used for localhost connections
  • Dispatcher URL: Base URL for your production/staging site
  • Organization ID & Program ID: Required for AEM Cloud Console links (find these in your Cloud Manager URL)

Tip: Create separate projects for local development, staging, and production environments.
